**标题设置**

```
一级标题：内容前加 #
二级标题：内容前加 ##
三级标题：内容前加 ###
```

**加粗和斜体**

```
加粗：加粗内容两端加 **
斜体：斜体内容两端加 *
```

**引用**

```
引用：引用内容前加 >
```

**列表和加框**

```
列表：列表内容前加 * 或 -
加框：加框内容两端加 `
```

**插入图片**

```
图片：! 加 [图片内容] 加 (图片网址)
```

**插入链接**

```
链接： [文字] 加 (链接网址)
```

**邮箱地址**

```
邮箱： <邮箱地址>
```

**删除线和分割线**

```
删除：删除内容两端加 ~~
分割线：三个或以上的 * 或 -
```
