## Inputting Special Characters

Why Alt + 9776 isn't working
When you type Alt + 9776, your computer isn't looking for the HTML code; it’s looking for a decimal Unicode value. However, standard Windows Alt codes (without a leading zero) usually reference the "OEM code page" (old DOS-style characters), which only goes up to 255.

When you type a number as large as 9776, Windows often "wraps around" or gets confused, resulting in it just typing the first character it recognizes—in your case, the ampersand (&).
